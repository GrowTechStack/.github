## 📌 About Project
**GrowTechStack**은 흩어져 있는 국내 IT 기업들의 기술 블로그 포스트를 수집하고, 개발자들이 성장에 필요한 인사이트를 더 쉽게 얻을 수 있도록 돕는 서비스입니다.

### ✨ Key Features
* **Auto-Collector**: 주기적으로 다양한 기업 기술 블로그의 RSS/Feed를 수집합니다.
* **Tech Curation**: 태그별, 기업별로 원하는 기술 트렌드를 필터링하여 확인합니다.
* **Daily Update**: 매일 아침 새로운 기술 아티클을 업데이트합니다.

## 🛠 Tech Stacks
* **Backend**: Java, Spring Boot
* **Data**: RDB (MySQL/PostgreSQL)
* **Infra**: GitHub Actions (CI/CD)
